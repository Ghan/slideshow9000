Bootstrapping
-----------------

* Create virtualenv and install Pyramid

::
    
    easy_install virtualenv
    virtualenv pyramid
    source pyramid/bin/activate
    easy_install Pyramid
    
* Run project

::

    source pyramid/bin/activate
    cd korppi
    python setup.py develop
    paster server development.ini
    
    
Enter development server

::

    http://127.0.0.1:6543/
